For the first section of this Case Study, I was expected to outline the purpose of security group rules in the diagram below. Also, to explain how these security groups are working and analyse the security that is provided to an organization that has data stored in the Database shown.



How I solved and explained the Purpose and how these Security Group works


The purpose of security group is to make sure that all the data kept in database is safe through security. Security groups work by specifying types of traffic to go through stopping unauthorized data to access the database, where you can be able to track and manage the data. As stated by the rules of the security group this feature controls the inbound traffic. In this case we can see at the diagram that the security groups rule is authorizing ports TCP 443 and port TCP 80 traffic from the web tier ELB. Also, it is authorizing inbound traffic in app tier ELB TCP ports 80 and 8080. Finally, Security groups is authorizing the app tier in port 3306 inbound traffic from app tier ELB. Security groups have a good level of protection but other levels of protection need to be taken in order to provide more layers of protection.


Case Study - Task #2


For the second section of this Case Study, I was expected to study the diagram carefully. You work in online retail company as a cloud engineer. Due to the current situation with COVID-19 they currently have high traffic demands to both their web servers (located on their public subnets) and their application internal servers (located on their private subnets). Currently, the online business is their only way of generating income, so it is imperative that their infrastructure is available at all times. They have already suffered multiple database outages and need your help. Their infrastructure supports modern application architectures including microservices and container-based applications. They also would like to be in a position whereby the system monitors the infrastructure and services traffic dynamically. They require 99.999% uptime.



Problem 1: The development team have asked you to review their current AWS architecture (as shown in the diagram above). They have asked you to explain in your own words recommendations that will improve the availability of the solution.

Problem 2: If there is a huge, continued spike in web traffic towards the App Servers over the Christmas period, explain and justify THREE inherently HA services the online retailer should take into consideration to help the overall availability of their application.

Check below the solutions that I gave to these issues


In regards to question 2, in section 1, there is a diagram where it shows the app servers and the database. Looking at the database we can see that there is a problem where if the database has an outage, there is no other one to work as a replacement. The solution in this case is to implement one of the database services from AWS called RDS Multi-AZ where there is a feature named automatic fail over and this has high availability with automatic database failover, which means if your database fails another one takes over with no data loss. In section 2, If there is a huge, continued spike in web traffic towards the App Servers over the Christmas period, then some improvements can be done to help the availability of their application. There are three cloud services with High Availability that can help which are Elastic Load Balancing, Amazon Cloud Front and Amazon Route 53. Together they provide better traffic scale and balancing, low latency and high transfer speeds as well as sufficient numbers of DNS services.
