The Key differences between storing data on S3 compared to store data on EC2 with EBS

In terms of storage when using S3 to store data in a company for example this resource is designed to store large amount of data and backup. On the other hand, the resources of storing data on EBS can be comparable to hard disks using in computers but virtualized. They differ in performance because ESB can have a better performance of S3. They both also have a high durability across different Availability Zones. The uses cases of Amazon S3 is that this resource can be used in Data Lake and big Data Analytics as well as Backup and restoration. On the other hand, EBS use cases are for Software Testing and Development, Business continuity and Enterprise-wide applications.
